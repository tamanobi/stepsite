# ステップサイト

Cyfons にインスパイアされたステップサイト

## 機能

* グループCRUD
* 会員CRUD
* 会員登録（グループ別）
* 記事CRUD（管理者のみ）
* 記事閲覧権限CRUD（グループ別）
* メール配信機能
* 管理者ログイン

## 迷っているところ

記事作成時のWYSIWYGライブラリ
